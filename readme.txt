=== Formulário condicional para pacotes (Contact Form 7) ===
Contributors: diogopetermann
Requires at least: 5.0
Tested up to: 6.5
Stable tag: 1.4

Campo condicional em formulários Contact Form 7. Exibe um campo de texto quando uma opção específica for selecionada em um campo SELECT.

== Descrição ==
Este plugin permite que campos de texto adicionais sejam exibidos em formulários do Contact Form 7 com base na seleção de opções específicas em campos SELECT. Ideal para formulários de orçamento de pacotes nacionais e internacionais, exibindo campos adicionais somente quando necessário.
